Magrathea Developer's Guide
===========================

The developer's guide is a mandatory source of information for anyone wishing to contribute
to Magrathea's development. This guide describes code standards as well as processes and tools
around the management of Magrathea's source code. Furthermore, it provides a comprehensive
documentation of Magrathea's API and its corresponding unit tests.
